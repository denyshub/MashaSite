# Інформаційна система військового округу

## Вимоги
- Node.js 
- npm 

## Інсталяція
1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/denyshub/MashaSite.git
   ```
2. Перейдіть у каталог проекту:
   ```bash
   cd site_mp
   ```
3. Встановіть залежності:
   ```bash
   npm install
   ```
4. Запустіть проект:
 ```bash
   cd military-district-is
   ```
   ```bash
   npm start
   ```
